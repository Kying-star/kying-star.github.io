## React起步

> 初始化react项目

```
create-react-app <yourProject Name>   //例create-react-app dome dome就是工程文件名
```

> 启动项目

```
npm start
```



> 推荐的项目目录

```
├─api                   // 网路请求代码、工具类函数和相关配置
├─application           // 项目核心功能
├─assets                // 字体配置及全局样式
├─baseUI                // 基础 UI 轮子
├─components            // 可复用的 UI 组件
├─routes                // 路由配置文件
└─store                 //redux 相关文件
  App.js                // 根组件
  index.js              // 入口文件
  serviceWorker.js      // PWA 离线应用配置
  style.js              // 默认样式
```
